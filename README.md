This is a task management system built with Next.js and Prisma.
Live URL: https://task-manager-nextjs-prisma.vercel.app/
## Getting Started

First, setup the environment variables in `sample.env` file and rename it to `.env`.
then init prisma and finally,
run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
